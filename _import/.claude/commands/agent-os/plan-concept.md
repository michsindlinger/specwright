# Plan Concept

Erstelle ein komplettes Bewerbungspaket aus einer Projektanfrage (RFP, Recruiter-Email, Kundenanfrage)

**Purpose:**
- Systematische Analyse der Anfrage und Anforderungen
- Domain-Recherche mit Perplexity/WebSearch
- Kompetenz-Mapping gegen das eigene Profil (Fit-Score)
- Konzept/Proposal-Erstellung mit USPs und Positionierung
- Pitch-Script fuer Interview-Vorbereitung
- Optionaler POC-Plan (--poc Flag)

**Output:** Bewerbungspaket in agent-os/concepts/YYYY-MM-DD-[project-name]/

Refer to the instructions located in @agent-os/workflows/core/plan-concept.md
