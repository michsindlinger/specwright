# Agent OS Web UI Mission (Lite)

Lokale Web-UI zur Steuerung von Claude Code mit drei Haupt-Views: Dashboard (Kanban), Chat Interface und Workflow Execution - basierend auf TypeScript Agent SDK und Lit Web Components.

Agent OS Web UI transformiert die CLI-Erfahrung von Claude Code in eine visuelle, intuitive Web-Oberfläche. Entwickler können ihre AI-gestützten Development-Workflows über ein modernes Dashboard verwalten, per Chat-Interface interagieren und komplexe Workflows ausführen - alles aus einer lokalen Web-Anwendung, die nahtlos mit dem TypeScript Agent SDK integriert ist.

**Wert & Differenzierung:** Local-first Architektur ohne Cloud-Abhängigkeiten, native Agent SDK Integration, moderne Lit Web Components, Moltbot-Style Dark Theme, und Echtzeit-WebSocket-Streaming für sofortiges Feedback bei Claude Code Operationen.
