# GSQ-004: Bottom Panel Grundstruktur

## User Story
Als Developer möchte ich ein ausklappbares Bottom Panel haben, das jederzeit sichtbar ist, damit ich die Queue und Specs unabhängig von der aktuellen View erreichen kann.

## Akzeptanzkriterien
- GEGEBEN die Applikation in beliebiger View
  WENN ich das Panel öffne
  DANN erscheint es am unteren Rand mit Slide-up Animation

- GEGEBEN ein offenes Panel
  WENN ich den Resize-Handle nach oben/unten ziehe
  DANN ändert sich die Panel-Höhe (min 200px, max 60vh)

- GEGEBEN ein Panel mit geänderter Höhe
  WENN ich die Seite neu lade
  DANN wird die gespeicherte Höhe wiederhergestellt

- GEGEBEN ein offenes Panel
  DANN sind 2 Tabs sichtbar: "Queue & Specs" und "Log"

## Technische Details

**Type:** Frontend
**Status:** Done
**Priority:** Critical
**Effort:** M (5 SP)
**Abhängigkeiten:** GSQ-001, GSQ-002, GSQ-003

### WAS
- Neue Komponente `aos-global-queue-panel`
- Light DOM mit injected Styles
- Fixed positioning, resize, tab navigation

### WIE
- Presentation Layer: Light DOM Component (wie `aos-cloud-terminal-sidebar`)
- `createRenderRoot() { return this; }` für Light DOM
- Static flag für einmalige Style-Injection in document.head
- CSS: `position: fixed; bottom: 0; left: var(--sidebar-width); right: 0; z-index: 999;`
- Open/Close: `transform: translateY(100%)` / `translateY(0)` mit `transition: transform 0.3s ease`
- Resize: Pointer-Events am oberen Handle (4px), mousedown → mousemove → mouseup
- Höhe in localStorage: `global-queue-panel-height`
- Tab-State in localStorage: `global-queue-panel-tab`
- Properties: `isOpen: boolean`, `activeTab: string`, `height: number`
- Events: `panel-close`, `panel-resize`, `tab-change`

### WO
- `agent-os-ui/ui/src/components/queue/aos-global-queue-panel.ts` (new)
