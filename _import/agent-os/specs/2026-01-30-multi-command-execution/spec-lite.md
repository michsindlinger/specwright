# Multi-Command Execution (Lite)

Tab-basierte parallele Ausführung von Workflow-Commands. Jeder Tab startet eigene Claude Code Instanz. User können mehrere Commands gleichzeitig ausführen und zwischen aktiven Executions wechseln, ohne laufende Prozesse zu unterbrechen.

**Key Features:** Tab-Navigation, Status-Indikatoren, Background-Notifications, Cancel-Logik
