# Spec Lite: Cloud Code Terminal

Ein integriertes Cloud Code Terminal als Sliding Sidebar im Agent OS Web UI. Entwickler können mehrere Claude Code CLI-Sessions direkt aus der Web UI starten, mit Modell-Auswahl aus allen konfigurierten Providern. Sessions bleiben über Page-Reloads und Projektwechsel hinweg erhalten.

**Key Features:**
- Terminal-Start-Button im Header
- Modell-Auswahl aus allen Providern
- Sliding Sidebar mit Multi-Tab Support
- Vollständige Claude Code CLI Integration
- Session-Persistenz über IndexedDB
- Projekt-Kontext Management

**Stories:** 9 (6 Feature + 3 System)
**Estimated Effort:** ~20 SP
