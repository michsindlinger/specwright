# Spec-Lite: Mobile View für Dashboard, Cloud Terminal & Spec-Detail

Mobile-responsives Layout (`<768px`) für Specwright UI: Dashboard (Variante A · Standard), Cloud Terminal (Variante D · Main) und Spec-Detail. Umgesetzt als additive `renderMobile()`-Branches in den existierenden Lit-Views plus 22 neue `aos-mobile-*` Sub-Components rund um ein generisches `aos-mobile-sheet` Primitiv. Voice, Bell-Notifications und Drag&Drop sind explizit OUT-of-scope.
