# Wizard-to-Sidebar Migration

Wizard-Modal durch direkte Sidebar-Terminal-Integration ersetzen. Getting Started View zeigt zustandsbasierte Kacheln: keine Kacheln bei fehlender Installation/Migration, Planning-Kacheln ohne Brief, Standard-Kacheln mit Brief. Setup-Befehle laufen im Cloud Terminal der Sidebar mit Auto-Execute und Auto-Detection nach Completion.
