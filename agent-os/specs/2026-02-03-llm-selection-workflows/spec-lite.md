# LLM Model Selection for Workflows

**Status:** Completed (2026-02-04)

**Quick Summary:** Enable LLM model selection (Opus, Sonnet, Haiku, GLM) when starting workflows from Workflows Dashboard, Context Menu, or Specs Dashboard. Replace hardcoded Opus with dynamic model parameter flowing from UI → Gateway → Backend → CLI.

**Business Value:** Cost-efficient workflow execution with cheaper models, consistent UX pattern matching existing Story card model selection, backward compatible with default Opus.
