# Spec Docs Viewer Extension - Lite Summary

> Created: 2026-02-12
> Full Spec: @agent-os/specs/2026-02-12-spec-docs-viewer-extension/spec.md

Erweiterung des Spec Document Viewers im Kanban Board: Statt nur spec.md und spec-lite.md werden alle *.md Dateien im Spec-Ordner (rekursiv, inkl. stories/ und sub-specs/) angezeigt, bearbeitet und gespeichert. Markdown-Checkboxen sind interaktiv klickbar mit sofortiger Persistierung.

## Key Points

- Dynamische Tab-Generierung aus allen *.md Dateien (rekursiv)
- Gruppierte Tabs nach Ordnerstruktur (root | stories/ | sub-specs/)
- Alle Dateien editierbar und speicherbar
- Interaktive Markdown-Checkboxen mit Auto-Persistierung
- Minimal-invasive Erweiterung bestehender Komponenten

## Quick Reference

- **Timeline**: 4 Stories, ca. 1-2 Tage
- **Dependencies**: Keine externen Dependencies
