# Interactive Workflows (Lite)

Interaktive Workflow-Ausführung in der WebUI: Workflows (wie /create-spec) starten über Karten, Fragen erscheinen als klickbare Optionen im Chat, generierte Dokumente werden live im eingebetteten Docs-Viewer angezeigt. Schritt-Indikator zeigt Fortschritt, Cancel-Button ermöglicht Abbruch.
